Who has built what

Knowing which firmwares exist, what they actually do, and whether you can have them keeps you from chasing a release that isn't out — or assuming a feature works when it's still in someone's private branch. State as of the 2026-05-18 snapshot.

◆ Snapshot caveat

Several of these were "unreleased, coming soon" at synthesis time. Any of them may have shipped since. For current status, ask in the Discord rather than trusting this table.

▲ Update (June 2026): public audio firmware now exists

The skill's 2026-05-18 snapshot said no public, buildable, audio-capable firmware existed. That is no longer true: chattock/sp1-tape-looper is a complete MIT-licensed four-track looper with source, two prebuilt .bins, and a loop-transfer web tool. It postdates the snapshot — read it as a worked example in the case study.

What you can build on today

Two public starting points now, for two different goals:

  • ericlewis/sp1-midi — the clean BSP foundation. A MIDI controller with working drivers for every peripheral and a tidy state-machine skeleton. Best when you want to start minimal and add the audiothingies/storagethingies reference engine yourself.
  • chattock/sp1-tape-looper — a complete, working audio firmware. Best when you want to see a real, end-to-end answer to "how do I get audio out of this thing": codec/I²S bring-up, a single-owner flash streamer, and a purpose-built on-disk layout. Read it as the case study.

Everything else with audio is still either TE's private IP or someone's unreleased branch.

What two recent authors learned (so you don't relearn it)

emvee1968 — basic playback in ~a week with Claude Code

  • Works: custom-stem playback, a gate effect, Bluetooth pairing (vol+/vol− combo), half-speed tape via function+play.
  • Doesn't: no power-off (had to disconnect the battery to re-enter the bootloader); no FF/RW (gave up after hours); only one of the stock effects.
  • Lesson: the power-off and FF/RW problems are real and non-obvious. Plan a clean shutdown path early. Discord #firmware, 2026-05-08

virtualflannel_46386 — most ambitious unreleased OS

  • Goes beyond stock: beat repeat and bit crusher aren't in the original TE effect rack — these are new modules.
  • Status: waiting for the web flasher to come back before releasing firmware + docs. Discord #general, 2026-05-11
